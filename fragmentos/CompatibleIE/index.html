Después del 

<meta http-equiv="content-type" content="text/html; charset=UTF-8" />
y antes del 
<title>
.

Ubicamos el: 

<meta http-equiv="X-UA-Compatible" content="IE=edge; chrome=1" />

Este elemento hace que el navegador IE use su modo mayor posible y no entre en modos alternativos. Si hay marco Chrome para IE lo usará.

Código 


                            
                
                <!--HTML-->                                    
                <meta http-equiv="X-UA-Compatible" content="IE=edge; chrome=1" />
