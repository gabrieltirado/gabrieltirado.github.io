Error Posición Absoluta Firefox

Contenedor con posición absoluta dentro de una celda de una tabla con posición relativa

En IE y Chrome al poner la posición relativa al td, el contenedor con posición absoluta no se sale del td,
pero en firefox el contenedor absoluto se va al top de la página. :(

Solución:

Cómo el posicionamiento en el td es tan problematico. Envolvemos (wrapper) el contenido de ese td en un div y a este si le ponemos absolute

Código 


               
                
                <!--Javascript (jQuery)-->
                
                
                <script>                    
                    /* Envolvemos el contenido en un div y a su vez lo ponemos relativo */
                    this.parent().wrap('<div></div>').css({ position: 'relative' });

                    /* Ahora el elemento si puede ser absoluto */
                    this.css({ position: 'absolute', top: 0, left: 0 });
                </script>
                
                
                <!--HTML-->
                
                
                <!-- Error -->
                <table>
                    <tr>
                        <td style="position:relative;">
                            <div style="position:absolute">
                                <contents>...</contents>
                            </div>
                        </td>                        
                    </tr>
                </table>
                
                
                <!-- Solución -->  
                <table>
                    <tr>
                        <td>
                            <div style="position:relative;">
                                <div style="position:absolute">
                                    <contents>...</contents>
                                </div>
                           </div>
                        </td>                        
                    </tr>
                </table>
